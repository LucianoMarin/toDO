<h1>LISTAS DE TAREA </h1>
<label>Programa para generar lista de tareas</label>
<br>
<label>Esta aplicacion permite anotar y eliminar tareas, asignarle nivel de prioridad</label>
<br>
<br>
<label>Es un programa simple, que usa PATRONES MVC, DAO y paradigma POO </label>
<br>
<label>Su funcionalidad es tan simple como escribir una tarea en el campo TextArea, este lo va apilando en un VBOX</label>
<label>Las tareas se van almacenando en una base de datos local, creada con sqlite, por lo cual la informacion persiste a menos que el usuario lo elimine</label>
<br>
<br>
<h1>TECNOLOGIAS USADAS: </h1>
<ul>
  <li>JavaFX</li>
  <li>SQLITE</li>
</ul>

<h1>LENGUAJE DE PROGRAMACION: </h1>

  <li>Java</li>
</ul>
